# 悬浮窗使用说明

## 功能概述
- 显示当前所有活跃闹钟任务：名称/标识、剩余时间/触发时间、启用状态。
- 悬浮窗支持拖拽移动，并记忆最后位置与显示状态。
- 在 uTools 主题切换时自动适配暗黑模式。

## 显示/隐藏
- 打开插件进入设置页，在“悬浮窗”区域点击“显示悬浮窗”即可开启；再次点击切换为“隐藏悬浮窗”。
- 悬浮窗右上角的“×”也可快速隐藏。
- 插件重启后会根据上次状态自动显示/隐藏。

## 位置记忆
- 在悬浮窗标题栏按住拖拽可移动位置（标题栏区域可拖拽）。
- 位置变化会自动保存到 `localStorage.floatingWindowState`，包含 `x`/`y` 坐标与 `visible` 状态。

## 数据刷新与性能
- 悬浮窗显示时每 500ms 刷新一次内容，隐藏或插件退出时自动停止刷新。
- 使用 CSS `transform: translateZ(0)` 开启 GPU 加速，保证刷新与拖拽流畅。

## 注意事项
- 悬浮窗采用无边框透明窗口，始终置顶但不影响 uTools 主窗口操作。
- 活跃任务依据时间块配置与全局开关进行计算，星期模式下仅在选中日期显示。

## 构建与发布
- 修改代码后在项目根目录执行 `./build.bat`，`dist` 目录将包含所有必要文件（包含悬浮窗组件）。
